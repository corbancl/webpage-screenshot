# 网页截图助手 - Chrome 插件

一款简洁美观的 Chrome 截图插件，支持多种截图模式。

## ✨ 功能特性

- 📸 **可视区域截图** — 截取当前屏幕可见内容
- 📄 **整页截图** — 自动滚动截取整个网页
- ✂️ **区域截图** — 鼠标框选任意区域截图
- 💾 **自动命名** — 按网页标题+时间戳自动命名保存

## 🚀 安装方法

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的**开发者模式**
3. 点击**加载已解压的扩展程序**
4. 选择本项目文件夹

## 📖 使用方法

1. 访问任意网页
2. 点击工具栏中的插件图标
3. 选择截图模式：
   - **可视区域截图**：直接点击"开始截图"
   - **整页截图**：选择后点击"开始截图"，自动滚动截取
   - **区域截图**：选择后点击"开始截图"，在页面上框选区域
4. 截图自动保存到下载文件夹

## 🛠️ 技术栈

- Manifest V3
- Chrome Extensions API
- OffscreenCanvas API
- 纯 JavaScript（无框架）

## 📝 注意事项

- 不支持 Chrome 内部页面（chrome://）
- 整页截图在内容较多的页面可能需要几秒钟
- 区域截图按 ESC 可取消
